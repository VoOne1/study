Проект 31. 
		Создать абстрактный базовый класс именем своей фамилии ( Surname), записанной латиницей. 
Создать   3  производных друг от друга  класса (а первый – от базового) с именами измененного имени базового класса с суффиксами вида «_1» (описание класса в хидере/ определение в файле .cpp).
Создать в производных  классах приватные члены-данных   типов   list <int*>* и char[10] -  в первом,  float* – во втором,   double* – в третьем  и  инициализировать их произвольно.
Реализовать отдельный от иерархии класс базы данных DB, в который в приватную область поместить хранилище типа list < Surname * >.
В функции main создать два объекта типа DB:  db1 и db2 (добавьте в db1 по одному объекту каждого типа, а в db2  - по 2 объекта).
Измените значения в db1 произвольным образом (например через change).
Скопируйте только часть данных, принадлежащих базовому типу (т.е. только то, что находится в базовой части каждого объекта), из первого объекта во второй:
             db2 = db1; 
(if, switch, dynamic_cast  и  typeid  не  использовать ! )
Распечатайте  данные из  db1 и db2.
