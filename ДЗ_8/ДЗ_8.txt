Проект 32. 
		Создать неабстрактный базовый класс именем своей фамилии (Surname), записанной латиницей. 
Создать   2  производных друг от друга  класса (а первый – от базового) с именами измененного имени базового класса с суффиксами вида «_1».
Создать в производных  классах приватные члены-данных   типов   list <string*>* -  в базовом, list< float>*  – во втором,   double* – в третьем  и  инициализировать их произвольно.
Реализовать отдельный от иерархии класс базы данных template <class T> DB, который наследует  list <T>.
В функции main создать объект типа DB < Surname * >.  И затем добавить в него по одному объекту каждого типа.
Измените значения в db1 произвольным образом (например через change).
Создать еще один объект типа DB < Surname * > :
  DB < Surname * > db2 = db1; 
(if, switch, dynamic_cast  и  typeid  не  использовать ! )
И сделать невозможным копирование:
      db1 = db2; 
Распечатать  данные из  db1 и db2.
